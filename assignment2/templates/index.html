🏆 Model Comparison & Analysis

Why all 3 models got 78.43% accuracy?

The dataset is small (253 images total) which limits all models equally. With limited data, simpler models like Naive Bayes perform as well as complex ones like KNN and Decision Tree.

How to improve accuracy:
✅ KNN — Try different K values (3, 7, 11) and use color images instead of grayscale
✅ Decision Tree — Limit max_depth to avoid overfitting, add more training data
✅ Naive Bayes — Apply PCA to reduce noise before training

Winner: KNN is preferred as it is most interpretable and stable for image classification with small datasets.
